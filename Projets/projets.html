Projet 1 : Interview Vidéo

Audiovisuel & Storytelling

Contexte : Réalisation complète d'une capsule vidéo type "Reportage" sur la passion de la musculation, tournée en conditions réelles.

  • 🎯 Le Défi : Capturer l'authenticité d'une séance de sport sans déranger les usagers, tout en gérant une acoustique difficile (bruit ambiant, musique de salle).
  • 🛠️ La Solution :
    • Pré-production : Négociation des accès et repérage technique pour identifier les zones calmes.
    • Tournage : Utilisation de micro-cravates pour isoler la voix et plans B-roll stabilisés pour dynamiser le montage.
    • Post-production : Montage rythmé sur Premiere Pro, étalonnage pour une ambiance "sport" contrastée.
  • ✨ Compétences clés : Gestion de projet, Relationnel client, Prise de vue/son, Montage (Adobe Premiere).

Projet 2 : Média Rap Français

Développement Front-End

Contexte : Création "From Scratch" d'un site média immersif dédié à l'actualité du rap urbain en 2024.

  • 🎯 Le Défi : Se démarquer des géants comme Booska-P avec une identité visuelle forte tout en garantissant un temps de chargement ultra-rapide.

Projet 3 : Marque Movado V

Stratégie Marketing & Branding

Contexte : Lancement fictif d'une marque de sneakers éco-responsables positionnée contre la Fast Fashion.

  • 🎯 Le Défi : Convaincre une cible Gen Z sceptique sur le Greenwashing et créer une communauté engagée autour de la marque.

    • Identité : Branding épuré et naturel inspiré par l'environnement urbain durable.

Projet 4 : Application Web Dynamique

Développement Back-End (PHP/SQL)

Contexte : Développement d'une plateforme musicale avec gestion d'utilisateurs et base de données complexe.

  • 🎯 Le Défi : Passer d'un site statique à une application dynamique sécurisée capable de gérer des milliers d'entrées (artistes, albums).
  • 🛠️ La Solution :
    • Architecture : Structure MVC (Modèle-Vue-Contrôleur) pour un code maintenable.
    • Base de Données : Modélisation MCD/MLD et requêtes SQL complexes pour lier Artistes/Albums/Utilisateurs.
    • Sécurité : Hachage des mots de passe (Argon2id), protection CSRF et injections SQL.
  • ✨ Compétences clés : PHP, MySQL, Cybersécurité, Architecture logicielle.
